===========
面向对象编程思想

类：具有相同特性和行为的一组对象的集合
java中编程的基本单元就是类
类的结构：
   public class 类名 {

       属性
       方法

   }
类就是由属性和方法组成
在代码中特性对应的是属性，行为对应的是方法.

声明属性：相当于声明变量

声明或定义方法语法:
public 返回值类型  方法名([形参列表]){
    方法体代码块
    [return 值;]
}
